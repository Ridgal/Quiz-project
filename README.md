# Webpack

### Установка:

Для установки на ваш компьютер нужно выполнить команду: 

- `git clone https://github.com/Ridgal/Webpack-template.git`
- `npm install`

### Команды:
Запуск сервера для разработки:
`npm run start`

Сборка проекта без оптимизации:
`npm run build:dev`

Сборка проекта с оптимизацией:
`npm run build:prod`

Очистка папки dist:
`npm run clear`
